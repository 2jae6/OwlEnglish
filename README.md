# Project Information
### 소개
 * 영단어를 검색하고 폴더별로 나누어 관리 및 시험을 볼 수 있습니다.
시험의 경우 모르는 것들만 시험에 다시 나오도록 설정합니다.

### 설치 
 * 별도의 설치가 필요없습니다.

### 사용 방법
 * Xcode 에서 빌드하신 후 바로 사용하시면 됩니다.

### 사용 언어 및 사용된 기술
  * Swift
  * SQLite3

### 실패 원인
 * 무료로된 사전 데이터를 구할 수가 없었고 구한 데이터베이스도 영단어사전이라기보다 백과사전에 가까운 데이터베이스였다.
